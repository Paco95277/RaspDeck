# RaspDeck

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.4.
This project consists primarily of a Raspberry Pi 5, a Raspberry Pi Pico, and a touchscreen. Its purpose is to create a Stream Deck-style device for programmers, allowing them to save various commonly used code snippets and command templates so they can quickly paste them into text fields on their PCs (tested in Windows 11) when needed.
The PCB was designed using Kicad. The housing is 3D-printed and was designed using Freecad.
